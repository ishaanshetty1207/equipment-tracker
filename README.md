# Equipment Tracker

A simple full-stack web application to manage equipment records.  
This project was built as a take-home assignment to demonstrate basic CRUD operations using React and Node.js.

---

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: JSON file (local storage)

---

## Features
- View equipment list in table format
- Add new equipment
- Edit existing equipment
- Delete equipment
- Search equipment by name

---

## Equipment Fields
- Name
- Type (Machine, Vessel, Tank, Mixer)
- Status (Active, Inactive, Under Maintenance)
- Last Cleaned Date

---

## How to Run Locally

### Prerequisites
- Node.js installed
- npm installed

---

### Backend Setup
```bash
cd backend
npm install
node server.js
